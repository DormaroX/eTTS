# eTTS Sandbox Directory

Dies ist die Test-Umgebung für eTTS-Entwicklung.

## Verwende die Sandbox für:

- ✓ Testen von Änderungen
- ✓ Experimenten mit neuem Code
- ✓ Integration-Tests

- ✗ Langzeitlagerung

Sie ist nur lokal für Tests gedacht.
